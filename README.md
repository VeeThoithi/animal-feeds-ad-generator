# Animal Feeds Ad Generator

## About
An AI-powered text and image ad generator for animal feeds that you can easily share across all platforms. Designed to help agrovet businesses grow by creating professional ads quickly and efficiently.

---

## Features
- Generates AI-powered text ads for animal feeds
- Generates AI-powered images for visual ads
- Easy sharing across social media and other platforms
- Simple frontend and backend structure for modular development
- Secure API integration using `.env` file

---

## Installation

1. **Clone the repository**
```bash
git clone https://github.com/VeeThoithi/animal-feeds-ad-generator.git
cd animal-feeds-ad-generator


 Install dependencies

# Backend
cd backend
npm install

# Frontend
cd ../ad-generator
npm install

Usage

Set up environment variables

Create a .env file in backend/ with your API key:

HUGGINGFACE_API_KEY=your_api_key_here


Run the backend

cd backend
npm start


Run the frontend

cd ../ad-generator
npm start


Open your browser at http://localhost:3000 (or your configured port)

Folder Structure
animal-feeds-ad-generator/
